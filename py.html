Python in Cybersecurity

1. Network Scanning

Python can be used to perform network scanning to find live hosts, open ports, and services. This is useful for reconnaissance in ethical hacking.

import socket

target = "192.168.1.1"
ports = [21, 22, 80]

for port in ports:
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    result = s.connect_ex((target, port))
    if result == 0:
        print(f"Port {port} is open")
    s.close()

2. Password Cracking

Python can automate brute-force attacks on login forms or hash cracking using libraries. Always use ethically on your own systems or authorized tests.

import itertools

chars = "abc123"
for length in range(1, 5):
    for guess in itertools.product(chars, repeat=length):
        print("".join(guess))

3. Packet Sniffing

Python can sniff network packets to monitor traffic, analyze protocols, or detect suspicious activity using libraries like scapy.

from scapy.all import sniff

def monitor(packet):
    print(packet.summary())

sniff(prn=monitor, count=10)

4. Hashing and Encryption

Python is used to hash passwords, encrypt data, and verify integrity, which is critical in cybersecurity.

import hashlib

password = "mypassword"
hashed = hashlib.sha256(password.encode()).hexdigest()
print("SHA-256 Hash:", hashed)

5. Automating Tasks

Python automates repetitive tasks like scanning multiple IPs, generating reports, or testing vulnerabilities.

import os

for i in range(1, 5):
    os.system(f"ping -c 1 192.168.1.{i}")
